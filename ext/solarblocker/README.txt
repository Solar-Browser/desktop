You can add your own list to here, but don't forget to edit "solarblocker.py.json" file too.

Example: If you add a list named "example.txt", then edit "lists" class in "solarblocker.py.json" like this;

1-"lists":[
        {
            "EasyList Cookie": true,
            "Fanboy's Notifications": true
        }
    ]

2-"lists":[
        {
            "EasyList Cookie": true,
            "Fanboy's Notifications": true,
            "example": true
        }
    ]
